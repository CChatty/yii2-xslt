# Yii-Wikipedia

### Installation

Begin by installing this package through Composer.

```js
{
    "require": {
        "thewulf7/yii2-xslt": "dev-master"
    }
}
```

```php
    [
        'components' => [
            'view' => [
                'class' => 'yii\web\View',
                'renderers' => [
                    'xslt' => [
                        'class' => 'thewulf7\yii2xslt\ViewRenderer',
                    ],
                ]
            ]
        ]
    ]                
```
